# Examples

This folder contains the smallest public proof artifacts for Q-Base by MVS.

- [`sample_project.json`](sample_project.json): minimal deterministic task batch used for the public evidence run.
- [`evidence_bundle/`](evidence_bundle/): collected run bundle, manifest, and run record generated from that sample project.

Use these files when you want a digestible example of the scheduler input and the resulting evidence output without walking the full runtime state.
