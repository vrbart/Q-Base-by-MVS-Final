# Gallery Upload Guide

Recommended upload order for the project page image gallery:

1. `qb_gallery_01_hero.png`
2. `qb_gallery_02_route_optimize.png`
3. `qb_gallery_03_evidence.png`
4. `qb_gallery_04_repo_flow.png`

These files were generated from the latest local demo artifacts in `dist/demo`.

Notes:

- They already use a 3:2 style layout that fits the project gallery well.
- If the platform compresses aggressively, prefer PNG uploads directly from this folder.
- Keep the gallery focused on orchestration, optimization, evidence, and reproducibility rather than general desktop screenshots.
