## Confirmed Requirements To Track

1. Submission is through Devpost for this event page.
2. Demo video duration must be within 2-5 minutes.
3. Public source repository is required.
4. Documentation/setup instructions must be included.
5. Technologies used and team details are required.

3. Record exact wording used in submission fields.
4. Ensure all required submission components exist and are linked.
