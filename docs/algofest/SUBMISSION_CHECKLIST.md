# AlgoFest Submission Checklist

## Required Assets

- [ ] Public repository link is live.
- [ ] Repository has `README`, setup instructions, and run commands.
- [ ] Demo video is uploaded and between 2:00 and 5:00.
- [ ] Devpost write-up sections are complete.
- [ ] Technologies used list is complete.
- [ ] Team details section is complete.
- [ ] Optional deck is attached (recommended).

## Repo Quality

- [ ] `ALGOFEST_SUBMISSION.md` is present and accurate.
- [ ] `START_HERE.md` contains a single happy-path run.
- [ ] No tokens/secrets are committed.
- [ ] Commands in docs have been run successfully from a clean shell.
- [ ] Evidence artifacts exist in `dist/algofest/evidence/`.

## Demo Readiness

- [ ] Primary demo follows `docs/algofest/VIDEO_SCRIPT_4M20S.md`.
- [ ] Shot list is pre-captured.
- [ ] Backup clips/screenshots exist for each major step.
- [ ] Timing rehearsal confirms <= 5:00.

## Final Pre-Submit

- [ ] Links open in anonymous browser mode.
- [ ] Repo default branch/tag matches demonstrated code.
- [ ] All claims in write-up are verifiable in repo/video evidence.
- [ ] Deadline/timezone checked.
