# Project Page Details

## Best Video Link Strategy

For the embedded project video field, use an **unlisted YouTube** link.

Why:

- the project page explicitly supports YouTube/Vimeo/Facebook/Youku embeds
- GitHub is good for code, not for reliable embedded video playback
- YouTube is the safest option for judge playback on desktop and mobile

Recommended split:

- **Video demo link**: unlisted YouTube URL
- **Try it out link #1**: GitHub repo
- **Try it out link #2**: optional direct proof/demo page if you later host one
